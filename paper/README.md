# Manuscript Source

This subtree is optional and is not required for rerunning the experiments.

It exists only to keep the tracked manuscript source close to the experiment outputs cited in the paper.

## Intended tracked files

- Manuscript source files such as `.tex`, `.bst`, and `.bib`.

## Intended local-only files

- Compiled manuscript PDFs.
- Journal templates and template archives.
- Cover letters, checklists, and office documents.
- Local revision notes and LaTeX build artifacts.

If you are using this repository only for reproduction, you can ignore this entire directory.
